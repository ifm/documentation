## meeting protocol

### VW AGV
+ Holonom fahrende Fahrzeuge: daher 4 Kameras - an jeder Ecke / Seite eine Kamera
+ Kombination aus Personenschutz und Objektschutz: Objektschutz ab gewisser Höhe über Boden
+ Anwendung:
    + Matrerialfuss
    + Montageassistent
    + sicherer Personenschutz

+ Wunsch: Gabelzinke bei 3m Entfernung bei schnellen Geschwindigkeiten
    + Problematik: Totwinkel im Nahbereich

+ Forschung:
    + Auslieferung von Paketen / Briefe auf Campusgelände in Bruchsal
    + bisher reines Forschungsprojekt
+ Flottenmangement über Tochter von VW gelöst
    + SEW impelmentiert VDA 5050 als Auftrag 
    + Flottenmanegment auf über SEW verfügbar
+ Induktive Ladung über Movitrans von SEW: zw. Ladungsphase ca 5 min Verwendung
